# AI-Student-Marks-Agent

A simple Tool Calling Agent built with Python and Gemini Function Calling.

## Features

* Add student marks
* View student marks
* Tool Calling using Gemini 2.5 Flash
* Agent Loop implementation
* Dynamic tool execution

## Tech Stack

* Python
* Gemini 2.5 Flash
* OpenAI SDK
* python-dotenv

## Example

**Input:**

```
Add marks for kunal = 85
```

**Output:**

```
Marks added successfully.
```

**Input:**

```
Show marks
```

**Output:**

```
kunal : 85
```

## Learning Concepts

* Tool Calling
* Function Calling
* Tool Schemas
* Tool Execution
* Agent Loop

Created as a beginner Agentic AI project.
