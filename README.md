# IQ Agent

A simple Agentic AI project built with Python and Gemini API.

## Features

* Generates a logical IQ question
* Uses AI to answer the generated question
* Demonstrates a basic multi-step AI workflow

## Tech Stack

* Python
* Gemini 2.5 Flash
* OpenAI SDK
* python-dotenv

## Run

```bash
pip install -r requirements.txt
python main.py
```
