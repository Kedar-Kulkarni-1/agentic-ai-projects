# AI Agent Starter Kit using Ollama

A beginner-friendly AI Agent project built using the OpenAI Agents SDK and Ollama.

## Features

* Runs locally using Ollama
* Uses OpenAI Agents SDK
* No OpenAI API key required
* Demonstrates basic agent creation
* Easy foundation for multi-agent workflows

## Tech Stack

* Python
* OpenAI Agents SDK
* Ollama
* Llama 3

## Installation

### Clone Repository

git clone https://github.com/yourusername/AI-Agent-Starter-Kit.git

cd AI-Agent-Starter-Kit

### Install Dependencies

pip install -r requirements.txt

### Pull Model

ollama pull llama3

### Start Ollama

ollama serve

## Run

python app.py

## Example Prompt

Tell a joke about agentic AI
